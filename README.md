# E-Commerce Sales Analysis

Analyzed 99,441 e-commerce orders from Olist (Brazilian marketplace) to uncover revenue trends, top categories, regional insights, and customer segments.

## Tech Stack
Python · Pandas · Seaborn · Matplotlib · SQL (SQLite) · Jupyter Notebook

## Key Findings
- Total Revenue: R$ 19.5M across 99,441 orders
- Average Order Value: R$ 205.30
- Top Category: Bed & Bath Table (R$ 1.71M)
- Peak Month: November 2017 (Black Friday effect)
- Top Region: Sao Paulo — 37% of total revenue
- Top 18% customers drive 60% of total revenue

## Dataset
Olist Brazilian E-Commerce Dataset (Kaggle) — 99,441 orders, 9 tables
